# StageZero.ReverseProxy

A complete reverse proxy library built with YARP for StageZero

## Namespaces

- StageZero.ReverseProxy.Models
- StageZero.ReverseProxy.Services  
- StageZero.ReverseProxy.Extensions

## Features

- Dynamic subdomain routing
- Let's Encrypt SSL automation
- WebSocket support
- Hot-reload configuration
- Blazor admin UI
- Auto certificate renewal

## Quick Start

Files have been generated in:
- StageZero.ReverseProxy/ - Library files
- BlazorUIPages/ - UI pages to copy
